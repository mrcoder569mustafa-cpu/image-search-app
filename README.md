#  Photo Search Engine | Unsplash API + JavaScript

A responsive photo search engine using the **Unsplash API**. Built with **HTML**, **CSS**, and **JavaScript**.  
Allows users to search for images, browse categories, and download images directly to their device.

##  Features

-  **Real-time Image Search** using Unsplash API.  
-  **Download Images** directly to your downloads folder.  
-  **Trending Categories** like Nature, Technology, and Animals.  
-  **Load More Functionality** for infinite scrolling.  
-  **Clean & Modern UI Design** with hover effects.  
-  **Fully Responsive** (works perfectly on both mobile and desktop).  

---

##  Technologies Used

- **HTML5**  
- **CSS3** (Modern design with hover effects)  
- **JavaScript (ES6)**  
- **Unsplash API**  
- **Font Awesome Icons**
